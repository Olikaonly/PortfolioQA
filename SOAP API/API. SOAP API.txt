Работа с API Яндекс.Спеллер.
Формат сдачи: XML-файл проекта SoapUI

Задание 1
Напишите тесты для метода checkTexts. 
Они должны проверять как минимум два языка, коды ошибок 1, 2 и 3, 
один и несколько блоков текста (spel:text). 
Каждый блок текста должен состоять из одного или нескольких слов.
Для каждого теста обязательно должны быть assert-ы, которые проверяют:
* правильность текста (word, s)
* код ошибки.

Один тест проверяет один конкретный сценарий, 
проверка вообще всех случаев в одном тесте — это плохая практика.

Задание 2*
Добавить mock service для проекта.
Добавить несколько вариантов ответов на запросы.
Ответы на запросы должны выбираться методом QUERY_MATCH 
в зависимости от пришедшего запроса. 
Проверять можно по тексту или, например, по значению атрибута lang.
